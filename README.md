# Сессионный проект по курсу “Операционные системы семейства Unix”

## Проект выполнил студент группы М25-555 Хилалов Муслим

## Описание

Работа выполнялась на виртуальной машине с использованием программы Oracle VM VirtualBox. Операционная система, выбранная для работы на виртуальной машине - Fedora Workstation 44.

Перед началом работы виртуальная машина была настроена. Был выбран сетевой мост в качестве адаптера для обеспечения доступа к Интернету. Были созданы два виртуальных диска, необходимые для выполнения заданий проекта.

По итогу работы были загружены в репозиторий 13 файлов. Файл project_history.txt является объединением двух историй: истории изначального пользователя (см. в файле метку `# USER: muslim`) и истории пользователя mephi-admin (см. в файле метку `# USER: mephi-admin`).
